# Protocolo de medición de variables SMA-03

Cada variable se puntúa en escala 0-1 usando anclajes estandarizados.

## P_E (Presión Económica)
- 0.0: Sin costos hundidos, capital reversible.
- 0.5: Capital moderado comprometido, costos crecientes.
- 1.0: Costos masivos, capital irrecuperable, futuro financiero en riesgo.

## FD (Percepción de Crisis Ambiental)
- 0.0: Entorno tranquilo, tiempo para deliberar.
- 0.5: Presión elevada, algo de tiempo para aprendizaje.
- 1.0: Crisis existencial, amenaza inmediata.
