# Protocolo de puertas de decisión

La frecuencia de las puertas depende del nivel de II:

- II > 0.70 (ROJO): Puertas frecuentes (Días 0, 7, 14, 30)
- II 0.50-0.70 (NARANJA): Puertas estándar (Días 0, 30, 90)
- II < 0.50 (VERDE/AMARILLO): Puertas flexibles (Días 0, 30, 90, 365)

Cada puerta debe incluir:
- Revisión de avances vs. supuestos
- Decisión: Continuar, Acelerar, Pausar, Revertir
- Actualización de registro de decisiones
